# Medical Cost Prediction

A Machine Learning web application that predicts hospital/medical treatment costs based on patient attributes.

## Project Overview

This project uses a machine learning model trained on an insurance dataset to estimate medical expenses based on:

- Age
- Sex
- BMI
- Number of children
- Smoking status
- Region

The model is deployed as an interactive web application using Streamlit.

## Technologies Used

- Python
- Scikit-learn
- Pandas
- NumPy
- Streamlit

## Machine Learning Workflow

1. Data preprocessing using ColumnTransformer
2. Feature scaling with StandardScaler
3. One-hot encoding of categorical variables
4. Model training using Linear Regression
5. Pipeline creation for preprocessing + model
